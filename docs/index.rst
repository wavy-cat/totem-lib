Welcome to the wavy-totem-lib's documentation!
==============================================

This is the documentation for the wavy-totem-lib library. This library lets you generate textures for Minecraft's Totem of Undying based on a player's skin using Python code.

You can find the source code and the issue tracker on the `repository page <https://github.com/wavy-cat/wavy-totem-lib>`_ on GitHub.
